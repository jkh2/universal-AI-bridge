## 🚀 Quick Start

### Option 1: Direct Use (GitHub Pages)
1. **Visit**: `https://your-username.github.io/universal-ai-bridge`
2. **Select AI Participants**: Choose 2 AI platforms
3. **Add API Keys**: Enter your API credentials
4. **Set Conversation Rules**: Define topic and parameters
5. **Start Collaboration**: Watch AIs collaborate autonomously!

### Option 2: Local Development
```bash
git clone https://github.com/your-username/universal-ai-bridge.git
cd universal-ai-bridge
# Open index.html in your browser
# For CORS proxy, use: npx cors-anywhere
```

### API Key Setup
You'll need API keys from your chosen AI platforms:
- **OpenAI**: [platform.openai.com](https://platform.openai.com)
- **Anthropic**: [console.anthropic.com](https://console.anthropic.com)
- **Google**: [makersuite.google.com](https://makersuite.google.com)
- **X.AI**: [console.x.ai](https://console.x.ai)

## 📋 Example Conversation Flow

```
1. Human sets topic: "Sustainable energy innovation strategies"
2. System prompts AI-1 with structured introduction
3. AI-1 responds with initial analysis and ideas
4. System automatically prompts AI-2 with context + AI-1's response
5. AI-2 builds on AI-1's ideas with additional perspectives
6. Conversation continues autonomously for set number of exchanges
7. Human can observe, whisper context, or join as participant
8. System maintains quality and prevents topic drift
9. Full conversation exported with analytics and insights
```
